# Strapi blocks renderer for Svelte

## Work in progress
This project is currently a work in progress and not fully complete. Suggestions are welcomed!

## Installation

```bash
npm install @charlesshook/strapi-blocks-svelte-renderer --save
```

## Components
- **Blocks**
  - paragraph
  - heading
  - list
  - quote
  - code
  - image
  - link
- **Modifiers**
  - bold
  - italic
  - underline
  - strikethrough
  - code
